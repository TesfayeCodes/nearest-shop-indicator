Contains deployment, configuration, and database resources for Nearest Shop Indicator.

## Contents

- `.env` — Environment variable samples or configs
- `docker-compose.yml` — Docker Compose for multi-service orchestration
- `ci-cd/` — CI/CD pipelines (e.g., GitHub Actions configs)
- `env/` — Environment-specific configs (optional)
- `nginx/` — Nginx configs for proxy/routing/static hosting
- `sql/` — Database schema and seed data scripts
