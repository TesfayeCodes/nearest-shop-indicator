# Frontend

The frontend is the user interface for Nearest Shop Indicator.

## Suggested Structure

- `src/` — Application source code
  - `components/` — UI components
  - `pages/` — App views/pages
  - `utils/` — Utilities (API clients, etc.)
  - `assets/` — Images, icons, styles
- `public/` — Static files like favicon, robots.txt, etc.
- `package.json` — Project dependencies/config
